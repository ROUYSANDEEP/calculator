
# Blueprint: Calculator Web App

## Overview

A simple, modern, and user-friendly calculator web application.

## Design and Features

*   **Layout:** A clean, centered calculator layout.
*   **Display:** A prominent display to show the current input and result.
*   **Buttons:** A grid of buttons for numbers (0-9), operations (+, -, *, /), clear, and equals.
*   **Styling:** Modern styling with rounded corners, shadows, and a clear visual hierarchy.
*   **Responsiveness:** The calculator will be responsive and usable on different screen sizes.

## Current Task

*   **1. HTML Structure:** Create the calculator's HTML structure in `index.html`.
*   **2. CSS Styling:** Style the calculator in `style.css`.
*   **3. JavaScript Logic:** Implement the calculator's functionality in `main.js`.
